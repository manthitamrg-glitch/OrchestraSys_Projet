# SmartSyslab

## Description
ce projet simule un systeme informatique avec plusieurs outils :
- Bash/Linux
-Python
-Docker
-MySQL
-GitHub

Le projet a ete realise en groupe

---

## Partie 1 : Bash
creation d'un script shell pour executer des commandes Linux.

Fichier :
- admin_systeme.sh

Exemple :
```bash
echo "Bienvenue"
```

---

## Partie 2 : Simulation CPU en Python
Simulation FIFO avec plusieurs processus.

Fichier :
- Simulateur.py

Exemple :
- P1 -> 5 secondes
- P2 -> 3 secondes
- P3 -> 2 secondes

---

## Partie 3 : Docker et MySQL
Utilisation de Docker compose pour lancer MySQL.

Fichier :
- docker-compose.yml

Base de donnees :
- etudiants_db

Table :
- etudiants

commandes CRUD utilisees :
- INSERT
- SELECT
- UPDATE
- DELETE
